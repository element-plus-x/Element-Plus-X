# vue-element-plus-x Changelog

## 2.0.0-beta.0 (2025-02-24)

### Features

- 正式发布 2.0.0-beta.0 版本
- 新增 XSender 输入框组件
- 新增 Thinking 思考中组件
- 新增 ThoughtChain 思维链组件
- 新增 Conversations 会话管理组件
- 新增 Attachments 附件上传组件
- 新增 FilesCard 文件卡片组件
- 新增 Bubble 对话气泡组件
- 新增 BubbleList 气泡列表组件
- 新增 Welcome 欢迎组件
- 新增 Prompts 提示集组件
- 新增 ConfigProvider 全局配置组件
- 新增 useRecord、useSend、useXStream 等工具 Hooks

### Improvements

- 优化 peerDependencies 配置，明确 element-plus 依赖
- 完善类型定义

### Breaking Changes

- 最低支持 Vue 3.5.17+
- 最低支持 Element Plus 2.9.7+
